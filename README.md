## 📂 Repository Structure
* `/scrapers` - Sub-Team A (Web Scraping)
* `/ai-engine` - Sub-Team B (AI/ML Logic)
* `/chatbot` - Sub-Team C (React Widget)
* `/email-campaigns` - Sub-Team D (Automation)
* `/dashboard` - Sub-Team E (Admin UI)
* `/shared` - Common utilities



## 📏 SOP Naming Conventions
* **Files:** Lowercase with hyphens (e.g., `lead-scraper.py`)
* **Variables (Python):** snake_case (e.g., `lead_score`)
* **Branching:** All PRs must target the `dev` branch. `main` is for production only.

## 🔐 Security
* Copy `.env.example` to `.env`.
* **Never commit real API keys.**
